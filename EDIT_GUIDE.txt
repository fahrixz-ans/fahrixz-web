================================================================================
PANDUAN EDIT DATA ADMIN - FAHRI XZ STORE
================================================================================

## 1. NAMA WEBSITE
File: .env (Frontend)
- VITE_APP_NAME=Fahri Xz Store
- Ganti "Fahri Xz Store" dengan nama website Anda

## 2. LOGO WEBSITE
File: public/logo.png
- Ganti file logo.png dengan logo Anda sendiri
- Ukuran rekomendasi: 512x512px
- Format: PNG dengan background transparan

## 3. FIREBASE CONFIGURATION
File: .env (Frontend)
- VITE_FIREBASE_API_KEY=your_api_key
- VITE_FIREBASE_AUTH_DOMAIN=your-project.firebaseapp.com
- VITE_FIREBASE_PROJECT_ID=your_project_id
- VITE_FIREBASE_STORAGE_BUCKET=your-project.appspot.com
- VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
- VITE_FIREBASE_APP_ID=your_app_id
- VITE_FIREBASE_MEASUREMENT_ID=your_measurement_id

Cara mendapatkan:
1. Buka https://console.firebase.google.com
2. Buat project baru atau pilih project existing
3. Klik ikon gear (Settings) > Project Settings
4. Scroll ke bawah ke bagian "Your apps"
5. Klik icon web (</>)
6. Copy config ke .env

## 4. FASPAY PAYMENT GATEWAY
File: server/.env
- FASPAY_MERCHANT_ID=your_merchant_id
- FASPAY_MERCHANT_NAME=Your Store Name
- FASPAY_USER_ID=your_user_id
- FASPAY_PASSWORD=your_password
- FASPAY_SIGNATURE_KEY=your_signature_key

Cara mendapatkan:
1. Daftar di https://faspay.co.id
2. Verifikasi dokumen usaha (NPWP, SIUP, dll)
3. Tunggu approval (1-3 hari kerja)
4. Login ke dashboard Faspay
5. Copy kredensial ke server/.env

## 5. EMAILJS (Invoice Email)
File: .env (Frontend)
- VITE_EMAILJS_SERVICE_ID=your_service_id
- VITE_EMAILJS_TEMPLATE_ID=your_template_id
- VITE_EMAILJS_PUBLIC_KEY=your_public_key

Cara mendapatkan:
1. Daftar di https://www.emailjs.com
2. Buat Email Service (Gmail, Outlook, dll)
3. Buat Email Template
4. Copy Service ID, Template ID, dan Public Key

## 6. OPENAI API KEY (Frandsa AI)
File: server/.env
- OPENAI_API_KEY=sk-your_api_key

Cara mendapatkan:
1. Daftar di https://platform.openai.com
2. Klik API Keys
3. Create new secret key
4. Copy ke server/.env

## 7. BACKEND API URL
File: .env (Frontend)
- VITE_API_URL=https://api.yourdomain.com/api
- Untuk local: VITE_API_URL=http://localhost:3001/api

## 8. KONTAK & SOSIAL MEDIA
File: .env (Frontend)
- VITE_SUPPORT_EMAIL=your-email@gmail.com
- VITE_SUPPORT_PHONE=your-phone-number
- VITE_SUPPORT_WA_CHANNEL=https://whatsapp.com/channel/your-channel

File: src/components/layout/Footer.tsx
- Update link sosial media (Instagram, Twitter, GitHub)

## 9. SEO DEFAULT
File: public/index.html
- Update <title>, <meta name="description">
- Update favicon.ico
- Update og-image.jpg (1200x630px)

## 10. WARNA BRAND
File: tailwind.config.js
- Update warna violet sesuai brand Anda
- Default: violet-600 (#7c3aed)

## 11. KATEGORI PRODUK
File: Firestore Database > collections > categories
- Tambah/edit kategori via Admin Panel
- Atau langsung di Firestore Console

## 12. PRODUK
File: Firestore Database > collections > products
- Tambah/edit produk via Admin Panel
- Upload gambar ke Firebase Storage

## 13. VOUCHER
Kolom masukan kode voucher yang di buat admin
kode voucher tersedia di saluran WhatsApp

## 14. Login & registrasi
metode login menggunakan
- email
- nomor telpon
- google
- apple
- github
- Facebook

================================================================================
CATATAN PENTING (WAJIB DIBACA)
================================================================================

1. TIDAK ADA MOCK DATA
   - Semua data berasal dari Firestore (real database)
   - Tidak ada data palsu/hardcoded
   - Semua produk, order, user, transaksi = REAL

2. CONTOH LAYOUT
   - Semua contoh layout di atas hanyalah REFERENSI
   - Tidak wajib ditiru 100%
   - Bisa disesuaikan dengan kebutuhan
   - Yang penting: fungsionalitas dan UX

3. ADMIN PANEL TERPISAH
   - Admin panel dibuat di repository terpisah
   - URL: https://admin.fahrixz-store.com (contoh)
   - Login dengan akun admin (isAdmin: true)

4. AI FRANDSA
   - Menggunakan ChatGPT API (OpenAI)
   - Nama: Frandsa (bukan AI, bukan bot)
   - Hak akses penuh ke data produk, order, user
   - Otomatis membaca data terbaru dari Firestore
   - Bisa handoff ke admin manusia

5. SALDO CASH QUEST
   - HANYA bisa di-withdraw (penarikan)
   - HANYA bisa ditukar voucher promo
   - TIDAK bisa digunakan beli produk
   - TIDAK bisa top-up
   - SAAT WITHDRAW otomatis mengirimkan request ke panel admin untuk di setujui atau di tolak

6. PAYMENT GATEWAY
   - Menggunakan Faspay (bukan Doku)
   - Support: VA, E-Wallet, QRIS, Credit Card
   - Mode sandbox untuk testing
   - Mode production untuk live

7. EMAIL SERVICE
   - Invoice: EmailJS
   - Verifikasi: Firebase Auth (built-in)
   - Notifikasi: Firebase Cloud Messaging

8. SEO
   - Menggunakan react-helmet-async
   - Meta tags dinamis per halaman
   - Open Graph untuk sosial media
   - Canonical URL

9. KEAMANAN
   - API key & secret di backend (tidak di frontend)
   - Firebase Auth untuk autentikasi
   - Firestore security rules
   - HTTPS wajib

10. DEPLOY
    - Frontend: Vercel (vercel.json sudah disediakan)
    - Backend: VPS/Cloud (DigitalOcean, AWS, dll)
    - Database: Firestore (Firebase)
    - Storage: Firebase Storage
